# WickedAlice.github.io

Требуется сделать single-page web-страницу, которая посредством API одного из фотосервисов показывает содержимое альбомов пользователя. 

1) авторизация пользователя в фотосервисе
2) показ списка альбомов
3) показ изображений выбранного альбома (без автозагрузки)
4) отображение tooptip'а при событии mouse over для изображения
5) пока выбранного изображения (без выбора изображения в зависимости от
размера экрана)
6) показ выбранного изображения с выбором изображения в зависимости от
размера экрана
